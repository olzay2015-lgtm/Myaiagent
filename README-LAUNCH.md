## AI Agent Platform - Запуск и документация

### Запуск сервера

#### Способ 1: Автоматический
1. Откройте папку `ai-agent-platform`
2. Двойной клик на **`launch.bat`**
3. Не закрывайте черное окно!
4. Откройте браузер и перейдите на http://localhost:4001/

#### Способ 2: Ручной запуск
```cmd
cd ai-agent-platform\apps\api
"C:\Program Files\nodejs\node.exe" server-http.js
```

### Проверка работы

```cmd
curl http://localhost:4001/health
```

## Возможности платформы

### 1. Веб-интерфейс (http://localhost:4001/)

**Функции:**
- Выбор AI агента из списка
- Чат с агентом в реальном времени
- Отправка и получение сообщений
- Система расписания задач

### 2. Telegram бот

**Бот:** @Myaiagenteo_bot

**Команды:**
- `/start` - Начать работу с ботом
- `/help` - Показать список команд
- `/schedule` - Показать запланированные задачи

**Естественный язык для напоминаний:**
- "Напомни через 5 минут сделать X"
- "Напомни завтра в 9:00 написать маме"
- "Напомни в 15:30 о встрече"

### 3. Система расписания задач

Задачи можно создавать двумя способами:

**Через веб-интерфейс:**
1. Откройте http://localhost:4001/
2. Нажмите кнопку "📅 Расписание"
3. Введите сообщение
4. Введите ваш Telegram Chat ID
5. Выберите дату и время (или используйте быстрые кнопки)
6. Нажмите "Добавить задачу"

**Через Telegram:**
- Просто напишите боту: "Напомни через X минут/часов/дней..."

**Как получить Telegram Chat ID:**
1. Напишите боту @userinfobot в Telegram
2. Он пришлет ваш личный Chat ID

## API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/health` | Проверка здоровья сервера |
| GET | `/agents` | Получить список агентов |
| POST | `/chat/:agentId` | Отправить сообщение агенту |
| GET | `/schedule` | Получить список задач |
| POST | `/schedule` | Создать задачу |
| DELETE | `/schedule/:id` | Удалить задачу |
| POST | `/schedule/trigger` | Принудительно запустить планировщик |
| POST | `/telegram/set-chatid` | Зарегистрировать Chat ID |
| GET | `/telegram/info` | Информация о Telegram боте |

### Пример создания задачи через API

```bash
curl -X POST http://localhost:4001/schedule \
  -H "Content-Type: application/json" \
  -d "{
    \"message\": \"Напоминание\",
    \"scheduledTime\": \"2026-03-14T15:00:00Z\",
    \"chatId\": \"5238995969\"
  }"
```

## Устранение проблем

1. **Сервер не запускается:**
   - Проверьте, не занят ли порт 4001
   - Закройте другие процессы node.exe

2. **Telegram бот не отвечает:**
   - Проверьте токен бота в .env
   - Проверьте логи сервера

3. **Задачи не выполняются:**
   - Проверьте правильность Chat ID
   - Убедитесь, что сервер запущен
