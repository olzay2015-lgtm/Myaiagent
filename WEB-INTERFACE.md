# AI Agent Platform - Web Interface

Красивый веб-интерфейс для общения с AI агентами прямо в браузере!

## 🚀 Быстрый старт

### Вариант 1: Двойной клик
Просто дважды кликните на файл **`start-web.bat`**

Интерфейс автоматически откроется в браузере!

### Вариант 2: Командная строка
```bash
cd ai-agent-platform
start-web.bat
```

### Вариант 3: Ручной запуск
```bash
cd ai-agent-platform\apps\api
node src\index.js
```

Затем откройте в браузере: **http://localhost:4000/**

## 💻 Возможности интерфейса

### Основные функции:
- ✅ **Выбор агента** - переключайтесь между разными агентами
- ✅ **Чат в реальном времени** - мгновенные ответы
- ✅ **История сообщений** - все сообщения сохраняются
- ✅ **Адаптивный дизайн** - работает на телефоне и компьютере
- ✅ **Индикатор набора** - видно когда агент печатает
- ✅ **Красивый UI** - современный градиентный дизайн

### Горячие клавиши:
- `Enter` - отправить сообщение
- `Shift + Enter` - новая строка

## 🎨 Скриншот

Интерфейс включает:
- Шапку с выбором агента
- Область чата с сообщениями
- Индикатор "печатает..."
- Поле ввода с кнопкой отправки
- Аватары для пользователя и агента

## 🔧 API Endpoints

Сервер также предоставляет REST API:

```bash
# Получить список агентов
GET http://localhost:4000/agents

# Получить список скиллов
GET http://localhost:4000/skills

# Отправить сообщение агенту
POST http://localhost:4000/chat/:agentId
Content-Type: application/json

{
  "message": "Привет!"
}
```

## 📁 Структура

```
apps\api\public\index.html  # Web интерфейс
apps\api\src\index.js       # Сервер
start-web.bat               # Скрипт запуска
```

## ⚠️ Важно

- Убедитесь что порт **4000** свободен
- Поддерживаются браузеры: Chrome, Firefox, Safari, Edge
- Для работы требуется только Node.js (установлен)

## 🎯 Демо-данные

В интерфейсе доступны 2 демо-агента:
1. **Marketing Assistant** - помогает с маркетингом
2. **Therapy Bot** - эмоциональная поддержка
