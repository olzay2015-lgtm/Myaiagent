# WebSocket Server

Socket.IO server for real-time communication (chat, voice streaming).

## Getting Started

```bash
npm install
npm run dev
```

The server will start on port 3001 (or PORT env variable).

## Structure

- `src/handlers/` - Event handlers (chat, voice)
- `src/middleware/` - Authentication and validation
- `src/rooms/` - Room management logic
